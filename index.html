Prompt

2. Preliminary Feasibility Check

Prompt

Conduct a quick search on PubMed/Google Scholar. Are there enough studies? Are there too many? refine scope.

3. PROTOCOL

Prompt

Draft the PROSPERO protocol. Include rationale, objectives, and methodology.
RESOURCE: PROSPERO Registration

4. Search Strategy

Prompt

List your search strings (MeSH terms) for each database (PubMed, Embase, Cochrane).
RESOURCE: PubMed Advanced Search

5. Titles/Abstract Screening

Prompt

Screen initial results. Paste list of included IDs or JSON data of decisions.
RESOURCE: Screening Tool (Rayyan/Covidence)

6. Full Text Screening

Prompt

Review full texts. Document reasons for exclusion (e.g., Wrong Outcome, Wrong Population).
RESOURCE: Full Text Tool

Prompt

Fill out the PRISMA numbers: Identification, Screening, Eligibility, Included.
RESOURCE: PRISMA Template

8. Data Extraction

Prompt

Extract data variables: Author, Year, Study Design, N, Age, Interventions, Outcomes. (Use JSON for auto-table!)
RESOURCE: Extraction Sheet

9. Missing Data

Prompt

Log attempts to contact authors for missing data or how missing data was handled (imputation).

10. Manuscript Abstract

Prompt

Write the structured abstract: Background, Methods, Results, Conclusion. Strict word count.

Prompt
